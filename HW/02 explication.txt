1. В чём разница между BOM и DOM? 

	DOM – "дерево" обьектов страницы (документа), а BOM - объекты браузера (окружения).
	Мое понимание: DOM - содержимое, а  BOM - инструменты и свойства.

2. В чём принципиальная разница между выводом сообщений с помощью alert и console.log? 

	alert - выводит в модальное окно и останавливает выполнение.
	console.log - выводит в консоль и не остановливает выполнение кода.

3. Чем отличаются ошибки чтения от ошибок выполнения? 

	Ошибки чтения обнаруживаются до начала выполнения кода и не позволяют его запустить.
	Ошибки выполнения могут возникнуть во время выполнения кода после его успешного старта.
	Ошибки выполнения - логические. Их нельзя выявить без анализа значений данных и процессов.	

4. Всё ли верно? var m; var M; var 7e; 

	Имя переменной не может начинаться с цифры (var 7e; - ошибка).

6. Какой будет результат выражений? По возможности, объясните почему именно такой результат. 
	
	"23" + 20 + '' = ? 			"2320" 
						C текстом оператор '+' - конкатенация, 
						число 20 преобразуется в string. "23"+"20"+'пустая строка' 

	"1" + (20 + 3) = ? 			"123" 
						Сначала действие в скобках (20 + 3)=23, 
						потом преобразование в string "23" и конкатенация "1" + "23" 

	"a" / 5 = ? 				NaN  
						Оператор '/' приведет к преобразованию в number строки "a". 
						Number("a") преобразуется в NaN. (NaN / 5) вернет NaN.

	("str" || 5) > 0 || (10 && true) = ?	true
						Сначала действие в скобках 
						1. ("str" || 5) вернет "str" ("или" Оба true - вернет первое).
						2. (10 && true) вернет true. ("и" Оба true - вернет второе).
						3. ("str" > 0) вернет false. Преобразует "str" в number (NaN) и сравнит с 0.
						4. (false || true) вернет true. ("или" true - если хотя бы один true).
